About Me

Hi! My name is Md Murad Hossain. I am currently working as a Software Engineer at SEBPO. I am a Full Stack Web Applications Developer who has been professionally developing software for over four years. I work with up to date technologies such as HTML, CSS, Bootstrap, JavaScript, Angular, Java, Groovy, Spring Framework, Spring Boot, Spring Security and MySQL among others to developer efficient backend systems to power high traffic web applications. I've created fully responsive web applications which scale seamlessly to meet needs of users.

I am a fluent english speaker and am interested in creating beautifully engineered web applications to service the needs of businesses and users alike.
